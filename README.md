Math-Quiz-with-Levels
A C++ console application for an interactive math quiz game.
This project is designed to test and improve arithmetic skills by offering multiple difficulty levels and operation types.

How to Play
The user is prompted to choose:

1-The number of questions they want to answer.

2-The difficulty level (Easy, Medium, Hard, or Mixed).

3-The operation type (Add, Subtract, Multiply, Divide, or Mixed).

The game then generates random questions based on these selections.

Features
Customizable Game: Choose the number of rounds, difficulty, and operation type.

Multiple Levels:

Easy: Numbers from 1-10.

Medium: Numbers from 10-50.

Hard: Numbers from 50-100.

Mixed: Randomly selects a level for each question.

Multiple Operations: Practice Addition, Subtraction, Multiplication, Division, or a random mix.

Instant Feedback: The game tells you if your answer is correct or wrong immediately.

Color-Coded Results:

Correct answers are highlighted in Green.

Wrong answers are highlighted in Red.

Learning Tool: For every wrong answer, the game displays the correct solution.



Technologies Used

Language: C++

Core Concepts: Functions, Structs, Enums, switch statements, system() calls for color and screen control, Input/Output (iostream), and Random Number Generation (cstdlib, ctime).
Final Score: At the end of the game, a summary shows the total number of correct and incorrect answers.

Play Again: A game loop allows the user to play again without restarting the application.
